## Python Bridge

CRYENGINE 5.7 Sandbox includes an official Python Bridge, allowing editor extension via Python scripts. However, Crytek provides no official documentation for it. Unofficial API reference and usage docs are available here: [CRYENGINE-5.7-python-documents](https://github.com/z060142/CRYENGINE-5.7-python-documents)
