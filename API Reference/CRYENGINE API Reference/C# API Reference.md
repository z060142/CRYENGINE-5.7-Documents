## Content

This reference contains documentation about all functions and properties of the Core project of the C# Framework. Functions in this project provide high level access to the mid level API of [CryCommon](../CRYENGINE%20Engine%20Code/Engine%20Modules/CryCommon.md).

This API Reference is only for the CRYENGINE C# API. For more information about the C# API please refer to the [Microsoft.NET documentation](https://msdn.microsoft.com/en-us/library/mt472912(v=vs.110).aspx).
