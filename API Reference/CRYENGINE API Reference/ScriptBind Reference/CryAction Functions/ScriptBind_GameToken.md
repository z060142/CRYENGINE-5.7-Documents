## Content

### SetToken

Set the value of a game token.

```
GameToken.SetToken( TokenName, TokenValue )
```

### GetToken

Get the value of a game token.

```
GameToken.GetToken( TokenName )
```

### DumpAllTokens

Dump all game tokens with their values to the log.

```
GameToken.DumpAllTokens()
```
