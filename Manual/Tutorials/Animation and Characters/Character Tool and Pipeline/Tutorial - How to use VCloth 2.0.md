## Content

VCloth 2.0 is an attachment for the Character Tool that gives users vastly expanded options when it comes to placing accurately simulated cloth onto their characters. The initial setup in Maya uses vertex colors to constrain cloth vertices. After exporting the cloth as a skin file, the cloth can then be setup directly in the Character Tool.
