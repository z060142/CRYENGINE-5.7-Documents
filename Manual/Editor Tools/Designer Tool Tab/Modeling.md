##
Overview

The Modeling tool is part of the Designer tool and can be opened in three different ways:

-
Through
**
Tools -> DesignerTool -> Modeling
**
;

-
By right-clicking an existing Designer object and choosing
**
Edit
**
;

-
When there is already a shape in your level, by selecting that shape and choosing
**
Shape -> Edit
**
 in the
**
Operators
**
 section in the
**
Properties
**
 tool:

[Image: /docs/static/attachments/44967927]
By default all buttons in the DesignerTool are inactive unless a Designer type object is selected in the viewport. The exception here are the tools in the
**
Shapes
**
 panel, that allow users to directly create new shapes on the viewport.

*

*

The Modeling tool has plenty of options to make and edit solid shapes:
