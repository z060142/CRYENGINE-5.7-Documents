## Overview

With the tools available under Transform in the Advanced section you can...

![Image](https://www.cryengine.com/docs/static/attachments/28900742)

### Pivot2Bottom

Moves the pivot to the bottom of the solid and centers it.

### ResetXForm

When a Designer object has been scaled and rotated with the standard translation tools (the ones in the main UI of the Editor), clicking this option will set the new values as default.

Example: you create a cube and scale it to 1.3 times the original size and then rotate the cube by 25°. Now pressing this buttons will keep the scale and rotation, but will reset the back to 1.0 scale and 0° rotation.

### Export

Exports.obj,.cgf, or.grp files.

### Snap to Grid

When Snapping to Grid (see [Grid Snapping](../../../../CRYENGINE%20-%20Getting%20Started/For%20New%20CRYENGINE%20Users/CRYENGINE%20V%20Basics/Snap%20%26%20Alignment.md)) is activated, will snap the selected vertex, edge, polygon or pivot to the nearest grid point.

### Hide Polygon

Hides the selected polygon.
