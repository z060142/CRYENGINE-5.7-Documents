## Content

CRYENGINE's Lens Flare Editor can be used to create lens flare effects to enhance the atmosphere and the perception of light entities. With this tool, unique flare effects can be created, the parameters of the existing effects can be modified and they can be managed or stored easily.

For more information about the tool and its functionalities, please refer to the
[Lens Flare Editor](../Editor%20Tools/Lens%20Flare%20Editor.md)
 page.

For more information about the system that CRYENGINE uses regarding the lens flares, please refer to
[Optical Flare System](../Editor%20Tools/Lens%20Flare%20Editor/Optical%20Flare%20System.md)
.
