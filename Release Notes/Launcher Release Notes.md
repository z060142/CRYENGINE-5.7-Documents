## Overview

New improved versions of the CRYENGINE Launcher are introduced from time to time, and on the pages that follow users can learn about the changes and any other important update information regarding each CRYENGINE Launcher build.

When a new version of the Launcher has been released/is available, users will be informed via an on screen pop-up window, on login then the most up to date version of the Launcher is installed automatically.

Finally, users can also check for Launcher updates and the specific version of the Launcher they are running by clicking on the Gear icon (top left-hand corner) of the main Launcher window and clicking on either Check for Updates or the About options.
