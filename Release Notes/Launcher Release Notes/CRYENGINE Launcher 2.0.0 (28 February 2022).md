## Content

The newest Launcher features a completely overhauled UI with a totally revamped back-end structure and technology stack.

In addition, as [mentioned earlier this year](https://www.cryengine.com/news/view/changes-are-coming-to-the-cryengine-marketplace), the CRYENGINE Marketplace has been refactored as a free asset trading platform called the Asset Database – where contributors can share assets from all disciplines of game development for free with their fellow CRYENGINE Developers.

## Release Highlights

#### New Launcher Application

All existing Launcher features have been moved to a new application platform with a full UI refresh.

#### Asset Submission

It is now possible to submit assets to the Asset Database via the web browser as well as the Launcher.

#### Asset Editing

You can now edit asset details without having to go through the entire review process.

#### Asset Versioning

Asset creators may now upload multiple versions of the same asset. Different Engine versions, bug fixes and improvements for assets are also supported.

#### New Packaging and Distribution Back-end

The backend has been completely revamped with a new technology stack, allowing for more flexibility for future features.

For more information on how to use the Launcher, please refer to the [CRYENGINE Launcher Reference](../../Manual/CRYENGINE%20-%20Getting%20Started/Installing%20CRYENGINE/CRYENGINE%20Launcher%20Reference.md) documentation.

## Fixed Known Issues

- **New:** Log in authorization has been improved.
- **New:** Asset downloads in the Launcher can now be triggered from the user's web browser.
- **New:** Assets can now be marked as deprecated by asset creators.
- **New:** Assets can now be reported by users.
- **New:** Comments on news articles can now be reported by users.
- **New:** Enterprise form validation has been improved.
- **New:** The privacy policy check workflow has been streamlined for ease of use.
- **New:** Users can now customize project thumbnails in the Launcher.
- **Fixed:** Users not being able to change their username in some rare occasions.
- **Fixed:** Pagination on news comments not working correctly.
- **Fixed:** Users occasionally not receiving confirmation emails for their purchases.
- **Fixed:** Users not being able to subscribe to newsletters on their account page.
- **Fixed:** No success status even if users presented a valid proof of identity when first setting up a Vendor account.
- **Fixed:** Users received multiple confirmation emails when ordering free assets.
- **Fixed:** Modal windows displayed in the wrong position.
- **Fixed:** Account link not working in emails.
- **Fixed:** Issues when fetching the asset database.
- **Fixed:** Users not being able to share tutorials on Pinterest.
